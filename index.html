HTMX Examples

This is a proof of concept of the AJAX functionality I was looking for from the HTMX library. This library allows you to easily a load a page or specific elements from other pages on the same domain via AJAX. This page demsonstrates some simple, but powerful examples. This is the source page that the content is being loaded from. The source page is a full html page that can be navigated to and used and in a traditonal fashion, but can also be used to load specific, indivual content.

Populate Element(div) with Entire Page Content

This is using hx-get to GET the content and hx-target to select the HTML element that should be updated. This defaults to repacing with innerHMTL.

Populate Element(div) with Specfic Page Content

The is is still using hx-get to GET the content and hx-target to select the HTML element that should be updated. However, hx-select is also being used to select a specific element from the source page. Note the title tag is updated by default on each GET. To prevent that from happeing hx-swap="ignoreTitle:true" can be placed on the same element that the relative hx-get is on.

Populate Element(div) with Specfic Page Content and CSS Animation

This is using classes to tigger a CSS animation. When the content is loaded, the .replace-with-specific-element-css-animation .fade-in class combination becomes present and the CSS animation starts.

Populate Element(span) with Specific Inline Content

This is an example replacing content inline with a span. I am not sure what the use case is for this yet, but wanted to test it.
